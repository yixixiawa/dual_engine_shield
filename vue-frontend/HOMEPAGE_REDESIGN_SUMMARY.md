- **结果**: 访问根路径/时直接跳转到首页

### 4. ✅ 2D/3D切换功能
在首页添加了模式切换器，支持两种视图：

#### 3D模式（默认）
- 显示3D地球模型
- 可视化全球钓鱼地址分布
- 支持交互操作

#### 2D模式
- 显示平面图表视图
- 展示全球钓鱼地址分布图
- 下方显示地区排行表格
- 包含：
  - 地区名称
  - 钓鱼地址数量
  - 占比百分比和进度条

### 5. ✅ 今日钓鱼地址显示

首页底部新增"今日发现的钓鱼地址"卡片，包含：

#### 功能特性
- **搜索过滤**: 可按域名或URL搜索
- **刷新功能**: 手动刷新钓鱼地址列表
- **详情查看**: 点击操作列查看完整详情
- **展开详情**: 点击左侧箭头展开完整URL和检测信息

#### 显示的数据
| 字段 | 说明 |
|-----|-----|
| 域名 | 钓鱼网站的域名 |
| 威胁等级 | 极高/高/中/低 |
| 风险评分 | 0-100% 的评分 |
| 来源地 | IP所在地区 |
| 检测时间 | 检测的时间戳 |
| 操作 | 查看详情 |

#### 样本数据
- paypal-secure.xyz (极高风险)
- amazon-verify.com (高风险)
- bank-of-china.net (高风险)
- apple-id-verify.io (极高风险)
- google-account-secure.tk (中风险)
- microsoft-login.ml (高风险)

## 🔧 技术实现

### 组件结构
- **模板**: 完整的响应式布局
- **状态管理**: 使用ref管理各种状态
- **类型安全**: 完整的TypeScript类型定义
- **样式**: SCSS响应式设计

### 关键特性
1. **模式切换**: 使用el-segmented组件实现平滑切换
2. **动态数据**: 钓鱼地址数据可动态加载
3. **搜索功能**: 使用computed实现前端搜索过滤
4. **详情对话框**: el-dialog展示完整信息
5. **进度条**: 风险评分可视化显示

### 响应式设计
- 移动端 (< 768px): 布局堆叠
- 平板 (768px-1024px): 两列布局
- 桌面 (> 1024px): 完整布局
